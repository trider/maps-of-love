Base Technologies and Libraries

The app was created using html, css, and javascript with the following libaries:

  • JQuery 1.10.2
  • PhoneGap 1.9.0
  • Bootstrap 2.3.2

Mapping Libraries

The app uses OpenStreetMaps via the leaflet.js library and associated plugins:

  • leaflet.js
  • Leaflet.Elevation-0.0.1.min.js
  • leaflet.usermarker.js
  • L.Control.Locate.js
  • gpx.js

Support Libraries

The following libraries provide additional functionality:

  • bootstrap-button.js
  • d3.js
  • jquery.timer.js
