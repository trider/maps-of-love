This page explains how to use the Chrome browser to view the Maps of Love online demo version. The demo runs under Chrome on Windows. It should also work on the Mac OSX version of Chrome. To return to the home page click here.

Install the Ripple Emulator

The Ripple Emulator lets you display mobile web sites and cross-platform web applications in your browser. You can install it from the Chrome App Store.

Install the Ripple Emulator

  1. Launch the Maps of Love online demo.
  2. In the Chrome tool bar, click the Ripple icon.
  3. In the Ripple Mission control, click Enable.
  4. If requested, Choose PhoneGap 1.0
  5. In the Devices list, choose a device. I recommend Generic - HVGA (320X480).

Using the App

  1. Download the Offroad.gpx map file to your PC.
  2. In the Demo App's start page, choose the Off-road and Day 1.
  3. Click Display.

Use Location and Tracking

  1. In Ripple's Navigation Simulator, click the Choose File button.
  2. Locate your simulated position. In the Demo app, click the Show me where I am button.
  3. Use the Speed list, to select a playback speed and click Go.
  4. Click the toolar button to expand the tool bar.
  5. Click Start tracking button to track your position.
